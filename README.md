📝 Descripción del Proyecto
Amigo Secreto es una aplicación web interactiva que permite a los usuarios organizar sorteos de manera fácil y divertida. La aplicación permite agregar nombres de participantes, validar y formatear las entradas, y realizar un sorteo aleatorio con animaciones visuales atractivas.

Ideal para organizar intercambios de regalos, asignar turnos o simplemente divertirse con amigos y familiares.

🚀 Estado del Proyecto
✅ Completado y Funcional - La aplicación está terminada y todas las funcionalidades principales operan correctamente.

📦 Instalación y Uso
Opción 1: Usar la versión online
Visita https://tu-usuario.github.io/amigo-secreto

¡Comienza a usar la aplicación inmediatamente!

Opción 2: Ejecutar localmente
Clona el repositorio:

bash
git clone https://github.com/tu-usuario/amigo-secreto.git
Abre el archivo index.html en tu navegador o utiliza un servidor local:

bash
# Con Python
python -m http.server 8000

# Con Node.js (http-server)
npx http-server
Abre tu navegador y ve a http://localhost:8000
